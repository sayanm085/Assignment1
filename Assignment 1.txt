1.What is HTML?


->HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page.


2.What is CSS?
-> Cascading Style Sheets (CSS) is a coding language that defines how a website looks and is laid out. It's a fundamental part of web design, along with HTML. 


3.What is the difference between front-end and back-end development?
-> Front-end development focuses on creating user interfaces that are well-designed, responsive, and fully functional. Back-end development involves creating reliable architecture that supports front-end development.


4.What is JavaScript?
-> JavaScript is a scripting or programming language that allows you to implement complex features on web pages .


5.What is a responsive web design?
->Responsive Web Design is about using HTML and CSS to automatically resize, hide, shrink, or enlarge, a website, to make it look good on all devices.
